# 线程的状态
线程的5种状态：
1. 新建

2. 运行

3. 等待(waiting/timed waiting)

4. 阻塞

5. 结束

##线程的5种状态之间变化
- 调用线程的`start`方法后线程进入**就绪**状态，线程调度系统将就绪状态的线程转为**运行**状态，遇到`synchronized`语句时，由运行状态转为**阻塞**，当synchronized获得锁后，由阻塞转为运行，在这种情况可以调用`wait`方法转为**等待**状态，当线程关联的代码执行完后，线程变为**结束**状态。
